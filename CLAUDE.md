# UniXchange — Claude Instructions

## Project
UniXchange — campus buy/sell/borrow/gig platform.

## Branding
- App name: **UniXchange** (exact casing always)
- Never use: Circonomy, Serponomy, or any old name

## Stack
Next.js 15 + Supabase + Clerk + Razorpay + Tailwind

## File structure
- `src/app/(dashboard)/` — all main pages
- `src/app/api/` — all API routes
- `src/components/shared/` — shared components
- `schema.sql` — database schema
- `reset.sql` — data reset script
